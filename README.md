# GitHub Profile Finder 🚀

A simple web application built using HTML, CSS, and JavaScript that allows users to search GitHub profiles and compare two GitHub users using Battle Mode.

## Features

- Search GitHub users by username
- View profile details such as:
  - Avatar
  - Name
  - Bio
  - Join Date
  - Portfolio Link
- View latest 5 repositories
- Loading and error handling
- Battle Mode to compare two GitHub users
- Displays winner based on total repository stars

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub REST API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` using Live Server or open it in a browser.
